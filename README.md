# פרימיום אלקטריק — Premium Electric

חנות בוטיק מקוונת למוצרי חשמל יוקרתיים. נבנתה כדמו חי לבחינת כיוון ועיצוב.

## טכנולוגיות
- **React 18** + **Vite** — מהיר, מודרני
- **Tailwind CSS** — עיצוב יוקרתי עם פלטה מותאמת (זהב + שחור עמוק)
- **Framer Motion** — אנימציות חלקות לכל הסקשנים
- **Lucide Icons** — אייקונים אלגנטיים

## מה יש בדמו
- Navbar דביק עם בלר ואפקט גלילה
- Hero בעוצמה עם תמונת רקע, כרטיס מוצר צף ו-USP
- גריד קטגוריות עם hover שמלא חיים
- קולקציית מוצרים עם סינון דינמי
- סקשן Showcase לכוכב הקולקציה
- רצועת מותגים מובילים
- ביקורות לקוחות
- הרשמה לניוזלטר
- Footer מקיף
- תמיכה מלאה ב-RTL ועברית

## הפעלה
```bash
npm install
npm run dev
```

ואז http://localhost:5173

## בנייה לפרודקשן
```bash
npm run build
npm run preview
```

## מבנה
```
src/
  components/   — כל הקומפוננטות של הדף
  data/         — קטלוג מוצרים לדוגמה
  App.jsx       — הרכבת הדף
  main.jsx      — entry point
  index.css     — Tailwind + עיצובים גלובליים
```
